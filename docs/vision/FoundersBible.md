# Project Atlas — Founder's Bible

## Vision

Build an AI marketing employee for small businesses.

Project Atlas learns a business, identifies marketing opportunities, creates campaigns, prepares content, and helps execute marketing consistently.

## Working Product Name

MarketingOS

## First Use Case

An exotic used car dealer connects their website inventory.

The system automatically chooses one vehicle each week and creates:

- Social posts
- Email campaign
- SMS copy
- Ad copy
- Suggested schedule
- Campaign angle
- Call to action

## Product Principle

The user should not need to think like a marketer.

The system should answer:

> What would a great marketing employee do next?

## MVP

The MVP proves one workflow:

Website URL  
→ Inventory extraction  
→ Business Brain  
→ Weekly campaign recommendation  
→ Social and email content  
→ Human approval

## First Customer

Small businesses with marketable inventory or services.

Initial vertical:

- Exotic used car dealers

Future verticals:

- Realtors
- Restaurants
- Dentists
- HVAC companies
- Med spas
- Gyms
- Ecommerce stores

## Differentiation

Most tools help users create marketing.

Project Atlas decides what marketing should be created next.

## Core Modules

- Business Brain
- Website Intelligence

- Publishing Engine

## Mantra

The best interface is not a blank prompt.

The best interface is a smart recommendation waiting for approval.
